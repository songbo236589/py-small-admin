# 功能特性

## 核心功能

### 用户认证与授权

- **JWT 认证**：基于 JWT 的无状态认证机制
- **双令牌机制**：访问令牌 + 刷新令牌，自动续期
- **角色权限**：基于角色的访问控制（RBAC）
- **菜单权限**：动态菜单渲染，权限精确控制
- **验证码登录**：图形验证码，防止暴力破解

### 用户管理

- **管理员管理**：管理员的增删改查
- **角色管理**：角色配置、权限分配
- **密码管理**：密码重置、安全加密
- **状态管理**：启用/禁用账号
- **批量操作**：批量删除、批量导入

### 系统配置

- **参数配置**：系统参数在线配置
- **邮件配置**：SMTP 邮件服务器配置
- **上传配置**：存储方式、文件限制配置
- **配置分组**：按业务分组管理配置
- **配置验证**：配置有效性测试（如邮件测试）

### 文件上传

- **多存储支持**：本地存储、阿里云 OSS、腾讯云 COS、七牛云 Kodo
- **文件类型**：图片、文档、视频、音频
- **图片处理**：自动生成缩略图
- **水印功能**：可配置图片水印
- **媒体库**：可视化的文件管理
- **安全控制**：文件类型、大小限制

### 日志系统

- **日志分级**：DEBUG、INFO、WARNING、ERROR、CRITICAL
- **日志输出**：文件 + 控制台
- **日志格式**：结构化日志，易于解析
- **日志轮转**：按大小/时间自动轮转
- **请求日志**：自动记录 HTTP 请求日志

## 量化数据管理

### 股票数据

- **多市场支持**：A股、港股、美股
- **实时行情**：最新价格、涨跌幅、成交量等
- **股票信息**：基本信息、财务数据、上市信息
- **数据同步**：定时从 AkShare 同步数据
- **搜索筛选**：多维度搜索和筛选

### 行业数据

- **行业分类**：按行业分类管理股票
- **行业行情**：行业涨跌幅、领涨股票
- **历史记录**：行业历史数据追踪
- **数据同步**：定时同步行业数据
- **统计分析**：上涨/下跌家数统计

### 概念数据

- **概念分类**：按概念标签分类管理股票
- **概念行情**：概念涨跌幅、热门概念
- **股票关联**：股票-概念多对多关系
- **历史记录**：概念历史数据追踪
- **数据同步**：定时同步概念数据

### K线数据

- **多周期 K 线**：1分钟、5分钟、15分钟、30分钟、60分钟、日K、周K、月K
- **K线数据**：开高低收、成交量、成交额、换手率
- **分表策略**：按时间分表，支持海量数据
- **数据查询**：高效的数据查询接口
- **可视化**：支持图表展示

## 异步任务

### Celery 任务队列

- **任务调度**：基于 Beat 的定时任务调度
- **任务重试**：失败自动重试，指数退避
- **任务路由**：按队列路由任务
- **任务监控**：Flower Web 界面实时监控
- **任务日志**：详细的任务执行日志

### 内置任务

- **股票数据同步**：定时同步股票列表
- **行业数据同步**：定时同步行业数据
- **概念数据同步**：定时同步概念数据
- **股票-概念关联**：定时同步关联关系
- **邮件发送**：异步批量发送邮件

## 开发体验

### 代码生成

- **模块生成**：快速创建新模块
- **模型生成**：基于数据表生成模型
- **API 生成**：自动生成 CRUD 接口
- **前端生成**：自动生成页面和组件

### 开发工具

- **命令行工具**：数据库迁移、数据填充等
- **API 文档**：自动生成 OpenAPI 文档
- **代码检查**：Ruff（Python）+ Biome（TypeScript）
- **Git 钩子**：Husky + Commitlint

### 热重载

- **后端热重载**：Uvicorn 自动重载
- **前端热重载**：Umi 开发服务器热更新
- **配置热更新**：支持配置热更新

## 性能优化

### 后端优化

- **异步编程**：全异步 IO，提高并发性能
- **数据库连接池**：连接池管理，减少连接开销
- **Redis 缓存**：热点数据缓存
- **查询优化**：合理的索引策略
- **分页查询**：大数据量分页查询

### 前端优化

- **按需加载**：路由懒加载，减少首屏加载时间
- **代码分割**：自动代码分割
- **资源压缩**：Gzip 压缩
- **图片优化**：图片懒加载、WebP 格式
- **CDN 加速**：静态资源 CDN 分发

## 安全特性

### 认证安全

- **JWT 认证**：无状态认证，安全可靠
- **令牌刷新**：自动刷新，避免频繁登录
- **密码加密**：Bcrypt 加密存储
- **登录限制**：验证码防暴力破解

### 数据安全

- **SQL 注入防护**：ORM 自动防 SQL 注入
- **XSS 防护**：前端自动转义
- **CSRF 防护**：CSRF Token 验证
- **数据加密**：敏感数据加密存储

### API 安全

- **API Key**：额外的 API 认证机制
- **权限验证**：接口级权限控制
- **请求限流**：防止恶意请求
- **HTTPS**：生产环境强制 HTTPS

## 扩展性

### 模块化设计

- **插件式架构**：模块间低耦合
- **中间件机制**：灵活的中间件扩展
- **依赖注入**：基于依赖注入的服务管理

### 配置系统

- **多环境配置**：开发、测试、生产环境配置分离
- **环境变量**：支持环境变量覆盖
- **配置热加载**：支持配置热更新
- **配置验证**：Pydantic 自动验证

### 插件扩展

- **存储扩展**：支持自定义存储适配器
- **认证扩展**：支持自定义认证方式
- **任务扩展**：支持自定义异步任务
