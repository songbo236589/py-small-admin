# 图标资源说明

此目录需要包含以下尺寸的图标文件：

- **icon-16.png**: 16x16 像素
- **icon-32.png**: 32x32 像素
- **icon-48.png**: 48x48 像素
- **icon-128.png**: 128x128 像素

## 图标要求

1. 格式：PNG
2. 背景：透明或纯色
3. 风格：简洁、易识别

## 临时图标

在开发阶段，你可以：
1. 使用在线工具生成占位图标
2. 从 iconfont 等图标库下载
3. 使用 Figma/Sketch 等设计工具制作

## 推荐在线工具
